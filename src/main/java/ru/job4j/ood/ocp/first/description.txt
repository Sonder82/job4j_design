На данном примере опишем ситуацию, когда мы решили создать программу по подсчету площади различных фигур.
Начали с фигуры - прямоугольник.
Создали класс Rectangle с полями длина и ширина.
Создали класс AreaCalculator, в нем будет реализация подсчета площади фигуры.
В дальнейшем потребовалось добавить фигуру в программу. Эта фигура круг.
Создали класс Circle с полем радиус.
В классе AreaCalculator добавили метод по подсчету площади круга.
В этом примере класс AreaCalculator не закрыт к изменению.
При каждой необходимости расширения фигур происходит его модификация.
Это противоречит принципу OCP.
